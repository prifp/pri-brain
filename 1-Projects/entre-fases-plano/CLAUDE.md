# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Projeto

**Entre-Fases** - Guia digital para mulheres 35+ na perimenopausa, climaterio e pos-menopausa.

Ajuda mulheres a identificarem como estao se sentindo no dia e oferece orientacoes de alimentacao, movimento e rituais de autocuidado adaptados ao momento.

## Logica Central

- **Perimenopausa/Climaterio**: orientacoes baseadas na fase do ciclo menstrual
- **Pos-menopausa**: orientacoes baseadas no nivel de energia e sistema nervoso
