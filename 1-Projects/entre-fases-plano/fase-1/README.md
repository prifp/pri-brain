# Fase 1 - Validacao com Grupo Piloto

## Objetivo

Aprender com 10 mulheres se a proposta do Entre-Fases gera valor real no dia a dia, usando WhatsApp como canal centralizado.

## Estrutura da Fase

```
ONBOARDING              ROTINA DIARIA           APRENDIZADO
(1x por pessoa)         (todo dia)              (continuo)
     |                       |                       |
     v                       v                       v
Questionario            Check-in +              Planilha de
inicial                 Orientacao              acompanhamento
```

## Arquivos desta fase

- `questionario-onboarding.md` - Perguntas para conhecer cada participante
- `check-in-diario.md` - Estrutura do check-in e respostas
- `planilha-acompanhamento.md` - Como registrar e analisar dados
- `mensagens-modelo.md` - Templates de mensagens

## Metricas de sucesso

| Metrica | Meta |
|---------|------|
| Taxa de resposta ao check-in | >50% nos dias ativos |
| Participantes ativas na semana 4 | >6 de 10 |
| Feedback positivo espontaneo | Qualquer sinal |
| Pedido para continuar/pagar | Bonus |

## Duracao

4-6 semanas
