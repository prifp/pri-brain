# Checklist de Lancamento - Fase 1

## Antes de Comecar (preparacao)

### Participantes
- [ ] Listar 10 mulheres para convidar
- [ ] Enviar convite individual para cada uma
- [ ] Confirmar quem topou participar
- [ ] Meta: minimo 8 confirmadas

### Infraestrutura
- [ ] Criar grupo de WhatsApp ("Laboratorio Entre-Fases")
- [ ] Criar planilha de cadastro (Google Sheets ou Notion)
- [ ] Criar planilha de check-ins diarios
- [ ] Definir horario do check-in (sugestao: 7h-8h)

### Conteudo
- [ ] Revisar questionario de onboarding
- [ ] Revisar templates de resposta para cada estado
- [ ] Preparar orientacoes de MOVIMENTO (com Ed. Fisico)
  - [ ] Energia baixa
  - [ ] Energia media
  - [ ] Energia alta
  - [ ] Ansiosa/agitada
- [ ] Preparar orientacoes de ALIMENTACAO
  - [ ] Energia baixa
  - [ ] Energia media
  - [ ] Energia alta
  - [ ] Ansiosa/agitada
- [ ] Preparar RITUAIS (oleos essenciais, chas, autocuidado)
  - [ ] Energia baixa
  - [ ] Energia media
  - [ ] Energia alta
  - [ ] Ansiosa/agitada
- [ ] Preparar variacoes por fase do ciclo (opcional na primeira semana)

## Semana 0: Onboarding

### Dia 1-2: Convites
- [ ] Enviar convite para todas as 10
- [ ] Registrar quem aceitou

### Dia 3-4: Grupo e Onboarding
- [ ] Adicionar confirmadas no grupo
- [ ] Enviar mensagem de boas-vindas no grupo
- [ ] Enviar questionario individual para cada uma
- [ ] Registrar respostas na planilha de cadastro

### Dia 5: Preparacao final
- [ ] Revisar dados de todas as participantes
- [ ] Ajustar horarios de check-in se necessario
- [ ] Avisar no grupo que comeca na segunda
