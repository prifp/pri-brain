# Food Sinergy

Aplicacao pratica do conceito de sinergia alimentar - combinacoes inteligentes que potencializam a absorcao e os beneficios nutricionais.

## Estrutura

```
food-sinergy/
├── conceitos/                    → Base teorica do livro
│   ├── introducao-livro.md       → Resumo comentado da introducao
│   ├── capitulo-1-doencas.md     → Cap 1: Coracao, AVC, Cancer, Diabetes
│   └── sinergias-basicas.md      → Lista de sinergias praticas
├── receitas/                     → Aplicacoes praticas
│   └── jantar-atum-abacate.md
├── meal-prep/                    → Organizacao e planejamento
│   └── lanches-manha-criancas.md
└── README.md
```

## Principio Central

> "Combinacoes inteligentes, nao pratos perfeitos."

## Objetivo

Criar um repositorio de conhecimento sobre combinacoes alimentares que:
- Maximizam absorcao de nutrientes
- Produzem efeitos anti-inflamatorios
- Melhoram controle glicemico
- Protegem o sistema cardiovascular
